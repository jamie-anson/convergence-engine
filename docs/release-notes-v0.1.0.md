# Convergence Engine v0.1.0

Initial public release of Convergence Engine `v1-lite`.

## Highlights

- Python-first deterministic convergence engine
- Locked `v1-lite` runtime types and machine-readable JSON Schemas
- In-memory runtime storage abstraction
- Built-in serialization and runtime schema validation helpers
- Reference examples for spec refinement and meeting orchestration
- Full conformance and determinism-oriented test suite
- GitHub Actions CI for Python 3.11 and 3.12

## Included in this release

- `ConvergenceEngineV1Lite` public API
- Typed runtime models for run, state, proposal, evaluation, constraint, and trail entries
- Packaged `v1-lite` schemas for integration-time validation
- OSS project docs, contribution guidance, and release checklist

## Not included yet

- Multi-evaluator search
- Branching and rollback
- Batch proposal evaluation
- Constraint expiry or revalidation
- LLM evaluators
- Seam composition
- Observability layer
- CLI or UI distribution
